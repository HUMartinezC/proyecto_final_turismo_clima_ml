# Fuentes de datos para proyecto de turismo, clima y analítica

| Bloque | Fuente | Qué aporta | Prioridad |
|---|---|---|---|
| Demanda turística | INE – Encuesta de Ocupación Hotelera | Viajeros, pernoctaciones, estancia media, ocupación por CCAA, zonas y puntos turísticos | Alta |
| Demanda turística | Dataestur | API turística agregada con datos de alojamientos, transporte, gasto, sostenibilidad, etc. | Alta |
| Clima España | AEMET OpenData | Temperatura, precipitación, viento, observaciones y climatología oficial | Alta |
| Clima histórico global | Copernicus CDS / ERA5 | Datos horarios desde 1940, útiles para cruzar clima con turismo | Alta |
| Portal datasets | datos.gob.es | Catálogo con datasets del INE y turismo, con identificadores API | Media |

# Ideas de variables derivadas interesantes

## Variables climáticas
- Temperatura media semanal
- Número de días de lluvia
- Índice de calor
- Eventos extremos (olas de calor, tormentas)

## Variables turísticas
- Ratio ocupación / clima
- Variación interanual de visitantes
- Gasto medio por turista
- Saturación turística por zona

## Variables temporales
- Temporada alta/baja
- Festivos y puentes
- Semana del año
- Vacaciones escolares

## Variables espaciales
- Distancia a playas o aeropuertos
- Densidad hotelera
- Accesibilidad

# Ideas interesantes para explotar estos datos

- Predicción de ocupación hotelera
- Forecast de demanda turística
- Impacto del clima sobre el turismo
- Recomendador de destinos según clima y temporada
- Detección de anomalías turísticas
- Dashboard geoespacial interactivo
- Sistema de alertas para saturación turística
- Modelos de series temporales multivariantes
- Gemelo digital turístico de una región
